### 基础获取物品指令
> #### json语法格式
- 作者`不会`，请自行补充
> #### 给予自己命令方块
```python
/give @s command_block
```
> #### 给予自己结构方块
```python
/give @s stucture_block
```
> #### 给予自己屏障
```python
/give @s barrier
```
> #### 给予自己光明方块
```python
/give @s light_block
```
> #### 给予自己允许方块
```python
/give @s allow
```
> #### 给予自己拒绝方块
```python
/give @s deny
```
> #### 给予自己NPC弹
```python
/give @s spawn_egg
```
### 药水buff效果
##### 如果不想要无限时间，把99999改为你要的时间(单位：秒)
> #### 夜视 (无线持续时间)
```python
/effect @s night_vision 99999
```
> #### 急迫 (无线持续时间)
```python
/effect @s haste 99999
```
> #### 力量 1 (无线持续时间)
